# 🌌 Galactic Miner

Idle incremental space mining game — built as a PWA, compiles to Android APK via GitHub Actions.

## Play
https://YOUR_USERNAME.github.io/galactic-miner/

## Build Android APK
Push to `main` branch → GitHub Actions auto-builds APK.  
Download from Actions → Artifacts.

## Features
- 6 planets with unique resources & mechanics
- Research tree (8 permanent upgrades)
- 12 upgradeable tools + 6 planet-specific upgrades
- Crystal Shop, Expeditions (15m-24h), Daily/Weekly Quests
- Prestige system with Dark Matter
- Asteroid events every 5 minutes

## Dev
```bash
python3 -m http.server 8080
# Open http://localhost:8080
```
